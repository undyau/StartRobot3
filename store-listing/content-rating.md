# IARC Content Rating Questionnaire — suggested answers

StartRobot3 is a timing/utility tool for orienteering event organizers. It has no game content, user-generated social content, or objectionable material, so the questionnaire should be straightforward.

| Section | Answer |
|---|---|
| App category | Utility / Reference (not a game) |
| Violence | None |
| Sexual content | None |
| Profanity / crude humor | None |
| Controlled substances (alcohol/tobacco/drugs references) | None |
| Gambling | None |
| User-generated content shared with others | None — start lists and recorded clips stay local to the device, never shared/uploaded |
| Shares user location | No — see [privacy policy](privacy-policy.md); location is used transiently for GPS time only, never shared or displayed |
| Digital purchases | None |
| Unrestricted internet access (e.g. browser, chat) | No |

Expected result: lowest available rating in every region (e.g. "Everyone" / "PEGI 3" / "3+").

Fill out the actual questionnaire in Play Console → your app → Policy → App content → Content ratings; this table is just the answer key.
