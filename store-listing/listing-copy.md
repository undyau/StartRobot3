## Full description (max 4000 characters)

```
StartRobot3 is a talking start clock built for orienteering event starts.

Run a fully automated start line with no announcer needed. StartRobot3 plays
timed audio cues — runner names from your IOF XML 3.0 start list, custom
recorded clips, text-to-speech announcements, and auto-generated beeps — all
locked to a fixed start interval (30 seconds, 1 minute, or 2 minutes).

KEY FEATURES

• Loads standard IOF XML 3.0 start lists and calls out each runner's name
  at their scheduled start time
• Mixes text-to-speech, your own recorded audio clips, and beeps into a
  single timed sequence
• Build custom "chains" of announcements anchored to the start or end of
  each interval, with as many chained cues before/after as you need
• Clock locks to wall-clock time and uses GPS time signals to stay accurate
  even with no phone signal in the field
• Nudge the whole schedule forward or backward by N minutes on the fly if
  the event falls behind or ahead
• Works fully offline — no internet connection required to run a start

WHO IT'S FOR

Orienteering, score-O, and similar interval-start events that need a
reliable, repeatable start announcer at one or more start points. Run one
device per course/start point.

PERMISSIONS

• Microphone — only used if you record your own announcement clips
• Location — only used to read GPS time for clock accuracy; your location
  is never stored, displayed, or shared
• Notifications — shows the required ongoing notification while the start
  clock is running
• Internet — only used if you load a start list from a URL instead of a
  local file

StartRobot3 does not collect or share any personal data. See the privacy
policy for details.
```
